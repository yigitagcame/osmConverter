# OSM Converter 


OSM data converter.


## Install

  ```

npm install

```

## Run

In development mode
```

npm start

```
or
```

npm run dev

```
In production mode
```

npm run prd

```

## Use

http://localhost:3000/{minLongitude}/{minLatitude}/{maxLongitude}/{maxLatitude}
  

## Test


```

npm run test

```
